Анализ структуры страницы сайта https://ru.wikipedia.org/

Для фона страницы используются белый и светло-серый цвет. 
Структура страницы: 
1. Header - содержит меню выбора действий (Переход на заглавную, авторизация, обсуждение, просмотр кода, история, поле поиска). Фон header светло-серый.
2. Content:
	a. Верхняя часть - Поле с приветствием и функцией создания новой статьи (светло-серый фон)
	b. Ниже контент разделён на две колонки и содержит несколько блоков информации, взятых в рамку (Избранная статья, Из новых материалов, Хорошая статья, Текущие события, Изображение дня, Этот день в истории)
	c. Под блоком статей расположено поле с информацией о связанных с википедией проектах.
3. Footer - содержит ссылки на политику конфиденциальности, описание википедии, обратную связь, мобильную версию сайта и пр. (светло-серый фон) 
4. С левой стороны страницы располагается вертикальное поле (светло-серый фон), содержащее логотип википедии с функцией перехода на главную страницу, навигация по сайту, ссылки на инструменты для работы с сайтом, ссылки на печать, ссылки на проекты, связанные с википедией, выбор языка отображения информации
